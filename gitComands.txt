git init - создать локальный репозиторий
git remote add origin (url репозитория) - соединяет локальный репозиторий с удаленным по url.
git clone (Url) - скачать файлы из удаленного репозитория.
git status - показывает текущий статус незакомиченных файлов.
git add <file> - подготавливает указанный файл к следующему коммиту.
git add . - все файлы директории 
git commit -m "string" - закоммитить (-m " " - добавить комментарий к коммиту)
git log - показывает историю всех коммитов в данной ветке.
git push origin <ветка> - загрузить данные последнего коммита на удаленный репозиторий.
git pull <имя репозиторя> <ветка> -скачать из данного удаленного репозитория все коммиты в данной ветке
которых еще нет в локальном репозитории.
git branch - выводит список локальных веток, доступных в данном локальном репозитории.
git branch <имя ветки> - создать новую ветку
git checkout <имя ветки> - переклюсает пользователя на данную локальную ветку.
все новые коммиты будут делаться в этой ветке.
git checkout <имя ветки с которой будет произведино слияние> -> git marge <имя ветки которую хотим слиять> 
(условно: 1 - master 2 - feature. 2ая ветка сольется с 1ой).
